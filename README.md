#### 项目介绍

---

这是由Spring Boot + Spring Data JPA + Hazelcast缓存 + JSONRpc + Swagger2 + Quartz + MySql + TYK + HTTPS + Junit等实现的一个金融管理系统，使用Maven模块化开发。

同时会有一些源码的深入和解析，这些技术点主要我都在我的博客中解释的较清楚，不足之处还请各位读者谅解。
